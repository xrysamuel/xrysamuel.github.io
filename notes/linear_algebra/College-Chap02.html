二、排列和逆序数

∣ Def ∣\color{011075}\left|\ \bf{Def}\ \right|∣ Def ∣ 排列

由 1,2,⋯ ,n1,2,\cdots ,n1,2,⋯,n 组成的有序数组称为一个 nnn 级排列，记为 j1j2⋯jnj_{1}j_{2}\cdots j_{n}j1​j2​⋯jn​

∣ Prop ∣\color{ed5bed}\left|\ \bf{Prop}\ \right|∣ Prop ∣

nnn 级排列的种数为 n!n!n!

∣ Def ∣\color{011075}\left|\ \bf{Def}\ \right|∣ Def ∣ 逆序数

各元素之间有一个标准次序，nnn 个不同的自然数，规定由小到大为标准排列

在一个排列 (j1j2⋯jt⋯js⋯jn)(j_{1}j_{2}\cdots j_{t}\cdots j_{s} \cdots j_{n})(j1​j2​⋯jt​⋯js​⋯jn​) 中，若 jt>jsj_{t} > j_{s}jt​>js​，则称这两个数组成此排列的一个逆序；否则称这两个数组成此排列的一个顺序

一个排列 j1j2⋯jnj_{1}j_{2}\cdots j_{n}j1​j2​⋯jn​ 所有逆序的总数称为此排列的逆序数，记为 τ(j1j2⋯jn)\tau (j_{1}j_{2}\cdots j_{n})τ(j1​j2​⋯jn​)

∣ Def ∣\color{011075}\left|\ \bf{Def}\ \right|∣ Def ∣ 排列的奇偶性

  • 逆序数为奇数的排列称为奇排列
  • 逆序数为偶数的排列称为偶排列

∣ Def ∣\color{011075}\left|\ \bf{Def}\ \right|∣ Def ∣ 对换

在排列中，将任意两个元素对调，其余的元素不动，该操作叫做对换；若将相邻两个元素对调，叫做相邻对换

∣ Thm ∣\color{4c7adb}\left|\ \bf{Thm}\ \right|∣ Thm ∣

一个排列中的任意两个元素对换，排列改变奇偶性

证明思路：首先证明相邻对换会改变奇偶性，再将任意对换分拆成奇数次相邻变换，于是便证明了任意对换都会改变奇偶性

∣ Cor ∣\color{ed5bed}\left|\ \bf{Cor}\ \right|∣ Cor ∣

  • 奇排列调成标准排列的对换次数为奇数
  • 偶排列调成标准排列的对换次数为偶数

∣ Cor ∣\color{ed5bed}\left|\ \bf{Cor}\ \right|∣ Cor ∣

n⩾2n \geqslant 2n⩾2 时，nnn 个元素的所有 nnn 级排列中，奇排列和偶排列的个数相等，各为 n!/2n!/2n!/2

证明思路：考察第一个位置和第二个位置，得到奇排列和偶排列的一一对应

∣ Ex ∣\color{b70203}\left|\ \bf{Ex}\ \right|∣ Ex ∣

证明 4×44 \times 44×4 数字华容道存在无法还原的初始情况，并出任意给出一种初始情况，它无法还原的概率

证明：横向移动：由于横向移动不改变数字序列，因此不改变逆序数
纵向移动：不论是向上还是向下，数字序列发生的改变都是「将一个数前移或后移三个位置」，会改变逆序数奇偶性
而目标的逆序数为 000，是偶数
所以首先可以将空格通过若干次对换挪至最下面一行，则后续还原过程中的纵向移动次数必为偶数次，也就是说，如果此时逆序数是奇数，那么无论如何也无法达到目标逆序数，如果此时逆序数是偶数，那么就可能可以通过若干次对换还原
若此时逆序数是偶数，则一定可以还原，证明过程可能会用到群论
