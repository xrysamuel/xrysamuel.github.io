C++入门

C++ 程序由一个或多个模块（函数）组成。程序从main()函数开始执行。函数由函数头和函数体组成。函数头指出函数返回值的类型和函数接受值的类型。函数体由一系列位于{...}内的 C++ 语句组成。语句有声明语句、赋值语句、消息语句、函数调用、函数原型、返回语句共6种语句。类是用户定义的数据类型规范，对象是更具类规范创建的实体。下面的程序展现了上述的内容：

// test.cpp
#include <iostream> // 预处理器编译指令，使用 iostream 文件的内容替换该指令
using namespace std; // 该语句使得程序可以使用名称空间 std 里的定义，也可以放到函数内

double ctof(double); // 用户定义函数原型


double ctof(double x){
    return 1.8 * x + 32.0; // 返回语句
}


int main(){
    double input; // 声明语句，声明一个 double 变量 input
    double output; // 声明语句，声明一个 double 变量 output
    cin >> input; // cin 是一个 istream 类对象，在头文件 iostream 内被定义，用 >> 运算符从输入流中抽取字符到 input 中 
    output = ctof(input); // input 作为参数传递给 ctof() 函数，它返回一个值，并赋给 output，ctof(input) 被称为函数调用
    cout << "The result is " << output << endl; // cout 是一个 ostream 类对象，在头文件 iostream 内被定义，
    cin.get(); // cin 是一个 istream 类对象，具有方法 get()
    return 0; // main() 返回语句
}

// myfirst.cpp -- displays a message

#include <iostream>                           // a PREPROCESSOR directive

int main() {                                 // function header
    // start of function body
    using namespace std;                      // make definitions visible
    cout << "Hello world!" << endl;           // message

    cout << "Comme up and C++ me some time."
         << endl;
    cout << "You won't regret it! " << endl;

    // If the output window closes before you can read it,
    // add the following code:
    cout << "Press any key to continue." << endl;
    cin.get();
    return 0;                                 // terminate main()
}                                             // end of function body

    cout << endl;

    carrots = carrots - 1;

}

// sqrt.cpp -- using the sqrt() function

#include <iostream>
#include <cmath>    // or math.h

{
    using namespace std;
   
    double area;
